### Immediate (Today)
1. **Install dependencies**
   ```bash
   cd FlowTipBot
   npm install
   npm install sha3
   ```

2. **Get API keys** (see QUICK_START.md)
   - Discord bot token
   - Flow wallet & private key
   - OpenAI API key
   - Firebase credentials

3. **Configure .env**
   ```bash
   cp .env.example .env
   # Edit with your keys
   ```

4. **Deploy & test**
   ```bash
   node src/deploy-commands.js
   npm start
   ```
